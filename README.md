# JS Test Examples

This codebase provides examples of how to use Jest and React Testing Library.

## Quick start

1. **Clone the `js-test-examples` repo**.

1. **Run the tests:**

   ```shell
   cd js-test-examples
   npm install
   npm run test
   ```
